# Research Intelligence Platform

A production-grade AI platform for research paper ingestion,
semantic search, retrieval-augmented generation (RAG),
literature review generation, and research assistance.

## Goals

- Upload and process research papers
- Semantic search
- Question answering over papers
- Multi-document reasoning
- Literature review generation
- Agent-based research workflows

## Tech Stack

- FastAPI
- PostgreSQL
- Docker
- SQLAlchemy
- pgvector
- PyTorch
- GitHub Actions
